
# Maldives Tuna — Industry Dashboard

This is a ready-made Next.js 14 project (TypeScript + Tailwind + Recharts) with minimal UI components to support your dashboard.

## Run locally
```bash
npm i
npm run dev
```

## Deploy
Push to GitHub, then import to Vercel — or use the Vercel CLI:
```bash
npm i -g vercel
vercel
vercel --prod
```
